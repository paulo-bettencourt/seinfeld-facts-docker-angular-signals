# Seinfeld Trivia

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.0.2.

This web application is a treasure trove of intriguing and entertaining facts dedicated to the beloved sitcom Seinfeld. Whether you're a die-hard fan or just starting your journey into the Seinfeld universe, this web app has something for everyone. Explore the rich history of Seinfeld and discover hidden gems that add depth and appreciation to your viewing experience.

## Technologies used

This GitHub repository contains a web application developed using the following technologies:

- Angular (Front-End): The application utilizes the latest Signals state management system and standalone APIs for seamless data management.
- Angular Material: Styling of the web application is achieved using Angular Material, providing a polished and consistent user interface.
- Docker: The Angular web application is containerized within a Docker image, ensuring easy deployment and portability.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

Run `docker build -t seinfeld-facts` to create the containerized image of the app in Docker.

Run `docker run -p 4200:4200 seinfeld-facts`to set the image on Docker.
